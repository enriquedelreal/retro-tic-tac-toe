# 🎮 Retro Game Arcade

A collection of classic retro games built with HTML5, CSS3, and JavaScript. Experience the nostalgia of classic arcade games with modern web technologies!

## 🎯 Live Demo

**[Play Now!](https://your-username.github.io/retro-tic-tac-toe/)**

## 🎲 Games Included

### Classic Board Games
- **Tic-Tac-Toe** - The timeless 3x3 grid game
- **Checkers** - Classic 8x8 board strategy game
- **2048** - Addictive number sliding puzzle
- **Minesweeper** - Classic logic puzzle with three difficulty levels

### Arcade Classics
- **Snake** - Navigate and grow your snake
- **Tetris** - Stack falling blocks to clear lines
- **Pong** - Classic paddle and ball game
- **Breakout** - Break all the bricks with your paddle
- **Pac-Man** - Navigate the maze and eat all the dots
- **Space Invaders** - Defend Earth from alien invasion
- **Battleship** - Strategic naval warfare game
- **Solitaire** - Classic card game

## 🚀 Quick Start

### Option 1: Play Online
Simply visit the live demo link above and start playing immediately!

### Option 2: Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/retro-tic-tac-toe.git
   cd retro-tic-tac-toe
   ```

2. **Open in your browser**
   ```bash
   # Using Python (if installed)
   python3 -m http.server 8000
   
   # Or using Node.js (if installed)
   npx http-server
   
   # Or simply open index.html in your browser
   ```

3. **Start playing!**
   Navigate to `http://localhost:8000` in your browser

## 🎮 How to Play

### General Controls
- **Game Navigation**: Use the sidebar to switch between games
- **Fullscreen**: Click the fullscreen button for immersive gameplay
- **Pause**: Press 'P' to pause/resume games
- **Reset**: Use the reset button to start a new game

### Game-Specific Controls

#### Pong
- **Player 1**: W/S or Arrow Up/Down
- **Mouse/Touch**: Move paddle with mouse or touch
- **AI Opponent**: Automatically controlled

#### Breakout
- **Movement**: A/D or Arrow Left/Right
- **Launch Ball**: Spacebar
- **Mouse/Touch**: Move paddle with mouse or touch

#### Pac-Man
- **Movement**: WASD or Arrow Keys
- **Objective**: Eat all dots while avoiding ghosts
- **Power Pellets**: Make ghosts vulnerable

#### Snake
- **Movement**: WASD or Arrow Keys
- **Objective**: Eat food to grow longer
- **Avoid**: Hitting walls or yourself

#### Tetris
- **Movement**: A/D or Arrow Left/Right
- **Rotate**: W or Arrow Up
- **Drop**: S or Arrow Down
- **Hard Drop**: Spacebar

#### Minesweeper
- **Left Click**: Reveal cell
- **Right Click**: Flag/unflag mine
- **Objective**: Clear all non-mine cells
